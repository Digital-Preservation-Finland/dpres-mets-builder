METS builder
============
A library for easy composing of METS files.

Usage
-----
Development
^^^^^^^^^^^
Create virtual environment and install requirements::

    python3 -m venv venv
    source venv/bin/activate
    pip install --upgrade pip
    pip install -r requirements_dev.txt
